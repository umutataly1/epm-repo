# epm-repo
The EPM (Evert Package Manger) Repository.
Evert (aka. epm) is a package manager for my distro called "Supernova linux".
This repo contains the registry file and the small files for the repo like;
Neofetch, Fastfetch, Nano, Vim, Curl, Flatpak and Git.

# Q&A
Q:Why is it called EVERT package manager?

A:When i was first making the distro it was named "Everest Linux" and i made
the package manager "EVERT PACKAGE MANAGER" but as it turns out theres already 
someone using that name and its dangeorus to use that so i changed it to Supernova
but i kept it evert for the honor of the old name.

Q:Why is epm made in bash but not C or Python?

A:For it to run on any machine (that uses tcc since i use that to compile c apps)

Q:Whats the first succesful package installed on epm?

A:Its neofetch, i installed it because i handnt set up the c compiler yet so i
installed neofetch since its purely made on top of bash, yes i also installed
bash onto my distro just because of neofetch.
